Phrase Structure Rules

CP -> TP | C TP
TP -> DP VP | DP T VP
DP -> DET NP
NP -> N | AP NP | NP PP
VP -> V | VP AP | VP PP | VP DP | VP DP DP | VP TP
PP -> P DP
AP -> A | AP A

C -> 'MD'
T -> 'MD'
DP -> 'NNP' | 'NNPS' | 'PRP' | 'WP'
DET -> 'DT' | 'PRP$' | 'WDT' | 'WP$'
N -> 'NN'
V -> 'VB' | 'VBD' | 'VBG' | 'VBN' | 'VBP' | 'VBZ'
P -> 'IN' | 'TO'
A -> 'RB' | 'RBR' | 'RBS' | 'WRB' | 'JJ' | 'JJR' | 'JJS'
