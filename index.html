Syntax Parser

English syntax parser based on the phrase structure rules for 80-180 Nature of Language.

Submitting the form will populate the token and syntax tree sections. Tokens use the Natural Language Toolkit convention.

Sometimes the part-of-speech tagger messes up. In that case, use the annotations field to indicate the correct tag: [ ["0", "DP"], ["2", "VB"] ]

In case of ambiguity, all valid syntax trees will be outputted.
