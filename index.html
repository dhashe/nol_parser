Syntax Parser

English syntax parser based on the phrase structure rules for 80-180 Nature of Language.

Submitting the form will populate the token and syntax tree sections. The Token section uses the Natural Language Toolkit convention, since that's the form that's actually used under the hood.

Sometimes the part-of-speech tagger messes up. In that case, use the annotations fields to indicate the correct tag. Indices are zero-indexed.

The phrase structure rules can be found here.

In case of ambiguity, all valid syntax trees will be outputted.
